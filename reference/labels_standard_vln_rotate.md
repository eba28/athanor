# Violin plot labels with horizontal x-axis text

Violin plot labels with horizontal x-axis text

## Usage

``` r
labels_standard_vln_rotate
```

## Format

An object of class `theme` (inherits from
[`ggplot2::theme`](https://ggplot2.tidyverse.org/reference/theme.html),
`gg`, `S7_object`) of length 7.
