# Map user-facing assay names to internal neighbor slot prefixes

Map user-facing assay names to internal neighbor slot prefixes

## Usage

``` r
map_assay_name(base_assay)
```

## Arguments

- base_assay:

  Character string; one of `"GEX"`, `"WNN"`, or an assay name passed
  through unchanged.

## Value

A character scalar prefix used to look up the neighbor slot.
