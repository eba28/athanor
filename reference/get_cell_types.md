# Pulls the info needed for `add_info_bar()` to a `DotPlot`

Pulls the info needed for
[`add_info_bar()`](https://eba28.github.io/athanor/reference/add_info_bar.md)
to a `DotPlot`

## Usage

``` r
get_cell_types(markers_df)
```

## Arguments

- markers_df:

  The markers data.frame filtered to match your input features

## Value

A data.frame with Cell_Type_Full and features.plot columns
