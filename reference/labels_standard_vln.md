# Standard label sizes for violin plots

Standard label sizes for violin plots

## Usage

``` r
labels_standard_vln
```

## Format

An object of class `theme` (inherits from
[`ggplot2::theme`](https://ggplot2.tidyverse.org/reference/theme.html),
`gg`, `S7_object`) of length 6.

## Details

Removes "Identity".
